# EpisodeSort

Setting that indicates how episodes are sorted for the show. (-1 = Library default, 0 = Oldest first, 1 = Newest first).


## Values

| Name                        | Value                       |
| --------------------------- | --------------------------- |
| `EpisodeSortLibraryDefault` | -1                          |
| `EpisodeSortOldestFirst`    | 0                           |
| `EpisodeSortNewestFirst`    | 1                           |
