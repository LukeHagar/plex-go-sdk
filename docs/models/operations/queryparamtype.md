# QueryParamType

The type of media to retrieve.
1 = movie
2 = show
3 = season
4 = episode
E.g. A movie library will not return anything with type 3 as there are no seasons for movie libraries



## Values

| Name                  | Value                 |
| --------------------- | --------------------- |
| `QueryParamTypeOne`   | 1                     |
| `QueryParamTypeTwo`   | 2                     |
| `QueryParamTypeThree` | 3                     |
| `QueryParamTypeFour`  | 4                     |
